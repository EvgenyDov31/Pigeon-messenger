# 🐦 Project Pigeon messenger

Консольный мессенджер, написанный на python, с использованием библиотеки socket

## 🚀 Стек технологий
* **Язык:** Python 3.14
* **База данных:** PostgreSQL, ORM - SQLAlchemy
* **Порт приложения:** 8080

### Запуск приложения
## 🖥️ Модуль сервера (`server.py`)

Сервер управляется интерактивно через консольные команды.

### Запуск сервера:
```bash
python server.py
```

### Доступные команды сервера:
* `start [флаг]` — инициализация и запуск сервера (по умолчанию `-a`).
* `stop` — остановка работы сервера.
* `clients [флаг]` — вывод списка всех подключенных в данный момент клиентов.
* `staccept` — запуск отдельного потока для приема входящих сетевых соединений.
* `kick [client_id] [reason]` — принудительное отключение клиента с указанием причины.
* `help` — вывод справки по командам администрирования.
* `exit` — корректное завершение работы серверного процесса.

---

## 📱 Модуль клиента (`client.py`)

Консольное приложение для пользователей системы, поддерживающее авторизацию, регистрацию и обмен сообщениями.

### Запуск клиента:
```bash
python client.py
```

### Доступные команды клиента:
* `connect` — установить соединение с сервером и запустить поток приема сообщений.
* `disconnect` — безопасно разорвать текущее соединение.
* `login [user_id] [password]` — авторизоваться в системе под своей учетной записью.
* `logout` — выйти из текущего аккаунта.
* `registration` — запустить интерактивный процесс регистрации нового пользователя.
* `send [user_id] [message]` — отправить сообщение пользователю по его ID.
* `loginStatus` — проверить текущий статус авторизации в сессии.
* `changePassword` — процедура смены текущего пароля.
* `deleteAccaunt` — полное удаление учетной записи из базы данных.
* `help` — список всех доступных пользовательских команд.
* `exit` — закрытие клиентского приложения.
